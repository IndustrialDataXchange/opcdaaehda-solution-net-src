# OPC DA/AE/HDA Client SDK .NET Standard

### Introduction
The OPC DA/AE/HDA Client SDK .NET Standard offers a fast and easy access to the OPC Classic Client technology. Develop OPC compliant DA, AE and HDA Clients with C#/VB.NET for the .NET Standard 2.1 framework.

The OPC DA/AE/HDA Client SDK .NET Standard API defines classes which can be used to implement an OPC client capable to access OPC servers supporting different specifications with the same API. These classes manage client side state information; provide higher level abstractions for OPC tasks such as managing sessions and subscriptions or saving and restoring connection information for later use.
